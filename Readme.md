# 🎭 Mood AI — CHAT_SHERA  An AI chatbot with 3 personality modes built with Streamlit + Mistral AI.  ## Modes - 😤 **Angry** — Hot-headed & savage replies - 😄 **Funny** — Jokes, puns & chaos   - 😢 **Sad** — Melancholic & moody vibes  ## Features - Switch moods anytime — chat memory stays - Rename the bot from chat itself - Responds in any language you write in - Fast streaming responses  ## Tech Stack - Streamlit - LangChain + Mistral AI (mistral-small-2506) - Python  ## Setup 1. Clone the repo 2. Install dependencies: `pip install -r requirements.txt` 3. Add `.env` file with `MISTRAL_API_KEY=your_key` 4. Run: `streamlit run UI_chatbot.py`
